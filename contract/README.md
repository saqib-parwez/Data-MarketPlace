# 🚀 Data Marketplace on Stellar (Soroban)

## 📌 Project Description

This project is a **decentralized Data Marketplace** built using **Soroban smart contracts on the Stellar blockchain**.

It allows users to:
- Upload datasets (or metadata like IPFS hashes)
- Set prices for their data
- Sell access to other users in a trustless way

Soroban enables secure, scalable smart contracts written in Rust and deployed on Stellar.

---

## ⚙️ What It Does

The smart contract provides the following functionality:

1. 📤 Data Upload  
   Users can upload data references (like IPFS hashes) and set a price.

2. 🔍 Data Discovery  
   Anyone can view available data listings.

3. 💰 Purchase Access  
   Users can buy access to datasets.

4. 🔐 Access Control  
   Only buyers can access purchased data.

## 📂 Contract Functions

| Function | Description |
|--------|-------------|
| `add_data` | Upload new dataset |
| `get_data` | Fetch dataset details |
| `buy_data` | Purchase access |
| `has_access` | Check ownership |
